About My Projects

Welcome to my projects section! Although many of these projects originate from FreeCodeCamp and might seem simple, they were designed to showcase a broad range of fundamental web development skills. Functionality may be important, but I believe that at its core, creativity and art - crafting engaging designs that captivate visitors is what drives web development. In building these projects, I tried to focus on responsive design and performance optimization. These projects and my work flow demonstrate my commitment to creating high-quality and user-friendly web applications, that are also highly maintainable. Each project was bundled with webpack alongside several plugins, Babel for compatibility, BEM for maintainable and scalable CSS and GIT for version control. The following are a list of features:

  • Webpack for bundling
  • Styling plugins such as Style Loader, CSS Loader
  • Optimization plugins such as Mini CSS Extract Plugin and CSS Minimizer.
  • Workflow plugins such as HTML Webpack Plugin and PostCSS
  • Babel translate ES6+ code for compatibility with older browsers and platforms.
  • BEM methodology in CSS.
  • GIT for version control

In addition, I wanted to showcase my budding skills as a digital marketer. Therefore, each individual project page has been designed not just as a web development piece, but as a product page. They feature a comprehensive product description, title, image, and call-to-action (CTA) links to the Github repo or the website (if applicable). This approach illustrates how I integrate digital marketing strategies into web development, creating a cohesive and engaging user experience.
